# mpeterson-personal-profile

A 2-3 page personal profile site built for Assignment 2 - Semantic HTML, CSS & Responsive Design. The site demonstrates a semantic HTML5 structure, accessible forms and images, modular CSS using Flexbox, and  responsive behavior at mobile and desktop breakpoints.

## Live site

## Structure
- index.html - Home/profile overview
- about.html - About/experience page
- contact.html - Contact form and details
- css/
    - base.css
    - layout.css
    - components.css
    - responsive.css
- images/ - image assets
- RATIONALE.md - design rationale
- LICENSE - MIT

## How to run locally
1. Clone the repo.
2. Open index.html in the browser
** Note: This project is mobile-first and uses semantic elements (<header>, <nav>, <main>, <section>, <article>, <footer>) and accessible form markup.
